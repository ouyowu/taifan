# 没有 AI key 时，后台每天怎么用

适用场景：
- 网站已经上线
- Supabase 已接通
- 还没有配置 AI key
- 当前以人工抓取、人工整理、人工发布为主

后台入口：
- `https://www.taifan.club/admin`

## 每天最短流程

1. 先看来源板
- 先检查这几条主来源：
  - `MEMINDY`
  - `Billkin Entertainment`
  - `PP Krit Entertainment`
  - `DOMUNDI TV`
- 依次点：
  - `先试抓`

2. 判断这条值不值得发
- 看 4 件事：
  - 标题是不是清楚
  - 链接是不是官方来源
  - 这条是不是新东西
  - 站里是不是已经有同一条

3. 值得发的，手动整理成动态
- 去后台动态区
- 新建一条动态
- 手动填写：
  - 标题
  - 中文摘要
  - 来源链接
  - 关联艺人
  - 分类

4. 有明确行程的，顺手补活动
- 如果来源里出现：
  - 时间
  - 城市
  - 场馆
  - 活动性质
- 就去活动区补一条活动

5. 检查前台
- 发布后看 3 个页面：
  - 首页
  - 动态页
  - 对应艺人页

## 每天发多少最合适

- 正常建议：
  - `2 到 5` 条动态
- 有活动周时：
  - `3 到 6` 条也可以

不要一口气发太多，先保证内容准确。

## 哪些最值得优先发

- 票务信息
- 官宣活动
- 品牌公开露出
- 预售 / 上新
- 中国粉丝最关心的行程变化

## 哪些先别发

- 标题太模糊
- 只有一句话、没有上下文
- 不是明确官方来源
- 重复内容

## 你现在不用做的

- 不用开 cron
- 不用跑自动 daily news
- 不用接 AI key

先把人工模式跑顺，再决定要不要自动化。
