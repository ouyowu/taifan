# ThaiStar Bridge / 泰娱桥 — 前台多彩改版复查 Brief

## 任务速览

- 本次任务：复查当前前台的最新视觉改版，并提出下一轮优化建议。
- 当前方向已经从“暗色编辑部风”切换到“更明亮、多彩、海报感更强”的路线。
- 请不要把这次任务理解成重做产品，而是判断这版是否已经更接近理想方向，以及下一轮该怎么继续打磨。

## 1. 当前目标

这次前台改版的目标非常明确：

- 参考 [Lattice](https://lattice.com/) 这类页面的多彩浅色块、轻盈卡片、粗体英文标题和明亮品牌感
- 同时保留 ThaiStar Bridge / 泰娱桥 作为“中文追星入口”的产品定位
- 加强明星宣传海报、艺人配图、活动海报在首页和分页里的存在感
- 把页面做得更像“内容 + 明星 + 活动”的现代可视化入口，而不是普通资料站

## 2. 当前已经完成的方向变化

### 视觉方向已经改变

和之前那版相比，当前前台已经不再是：

- 深色主站
- 珊瑚橙强调
- 偏 editorial dashboard

而是已经切换成：

- 米白 / 奶油白主底
- 多彩浅色块：淡紫、薄荷绿、浅蓝、浅粉、浅黄
- 大圆角内容块
- 大号粗体英文标题
- 海报型明星图、头像、配图更明显
- 更接近 Lattice 那种“浅色品牌内容站”感觉

### 页面里已经加入的内容形式

- 首页 Hero 已改成大渐变背景 + 多块悬浮明星内容卡
- 首页热门明星区已经不只是列表，而是大图卡片
- 活动列表已经补了明星海报图
- 动态列表已经补了明星配图
- 明星页已经强化成更像“艺人宣传页”的结构
- 服务页、攻略页、详情页也已经统一到多彩浅色板块风格

## 3. 不可破坏的产品规则

### 语言规则

必须保留：

- 界面、按钮、说明文案：简体中文优先
- 明星英文名、场馆名、商场名、品牌名、官方活动标题：原文保留
- 页面展示逻辑：`中文解释 + 原文保留`

不要强行把这些翻成中文：

- 明星官方英文名
- 泰文名
- 品牌名
- 场馆名 / 商场名 / Hall 名
- 官方活动标题

### 产品定位

不要把站点改成：

- 泛娱乐网站
- 娱乐八卦站
- 纯海报展示站

仍然要保留它的核心定位：

- 帮中国大陆粉丝降低追星门槛
- 强调中文整理能力
- 强调官方来源优先
- 强调活动 / 动态 / 明星 / 服务的入口关系

### 可信度表达

以下表达可以优化样式，但不要删：

- 官方来源
- 中文整理
- 原始标题 / 官方写法
- 来源说明
- 活动地点原文

3. 首页的悬浮内容卡，现在是“高级”还是“有点乱”？
