# ThaiStar Bridge / 泰娱桥 前台 UI 视觉优化 Brief

## 任务速览

- 本次任务：复查并优化前台 UI 视觉表现，不需要先改后台结构。
- 重点不是重做产品逻辑，而是提升视觉层次、统一感、可信感和首页可读性。
- 请优先查看本文第 11 节列出的页面文件，再判断哪些页面最值得先优化。

## 1. 项目定位

ThaiStar Bridge / 泰娱桥 是一个面向中国大陆粉丝的泰国娱乐信息聚合与服务平台 MVP。

当前核心目标不是“做一个很炫的媒体站”，而是：

- 让中国用户快速看懂泰国明星活动和动态
- 降低追星门槛
- 用中文解释复杂信息
- 但保留真正需要拿去搜索、买票、找地图、核对官宣的原始名称

## 2. 这次希望 Claude 做什么

请把这次任务理解成：

- 优化前台视觉体验
- 优化信息层次
- 优化页面之间的统一感
- 优化首页的信息过载问题
- 强化“可信来源”和“中文整理”的感知

不要把这次任务理解成：

- 推翻当前产品定位
- 重写全站信息架构
- 把页面改成通用 AI 审美模板
- 为了“设计感”牺牲中文可读性和实用性

## 3. 不可破坏的产品规则

### 语言规则

必须保留这套展示逻辑：

- 界面、导航、说明文案：简体中文优先
- 明星名、商场名、场馆名、品牌名、官方活动标题：尽量保留原文
- 页面展示方式：`中文解释 + 原文保留`

请不要把这些内容强行直译成中文：

- 艺人官方英文名
- 商场 / 场馆 / Hall 名称
- 品牌名
- 官方活动名
- 机构名

### 信任规则

页面里已经有明确的可信内容表达，请不要删掉，只能优化视觉表现：

- `官方来源优先`
- `已审核发布`
- `来源：xxx`
- `原始标题 / 官方写法`
- `中文摘要`
- `中文说明`

### 用户规则

用户主要是中国大陆粉丝，所以：

- 要让人一眼知道这页在讲什么
- 要让新粉不慌
- 要让“下一步去哪里看”非常清楚
- 不能为了设计感把页面做得过于抽象

## 4. 当前视觉方向

当前前台整体已经有一个比较明确的方向：

- 深色底
- 深海军蓝 / 墨蓝为主
- 暖珊瑚橙作为强调色
- 大圆角卡片
- 轻玻璃感 / 半透明感
- 偏“内容编辑部 + 运营看板”的气质

可以把它理解成：

`暗色 editorial dashboard + 中文内容服务入口`

这个方向本身是可用的，不需要完全推翻。  
更适合做的是：

- 提纯
- 收敛
- 拉开层级
- 减少重复卡片感
- 让几个重点区域更有“主视觉”和“呼吸感”

## 5. 当前最明显的问题

### 首页

首页已经承载了很多功能，但目前最容易出现的问题是：

- 信息很多，容易显得拥挤
- 新粉提示很多，局部有些碎
- 卡片数量偏多，视觉节奏容易趋同
- “最重要的入口”虽然都在，但第一眼主次还可以更强
- 热门明星、最近活动、最新动态、可信来源、各种提示区之间的视觉优先级还可以更明确

请重点考虑：

- 首页是否需要更明确的 `Hero > 核心入口 > 热门追踪 > 内容流` 层次
- 哪些提示应该合并
- 哪些卡片应该更轻
- 哪些区块应该更像“主模块”而不是普通卡片

### 活动日历页

当前方向是对的，但还可以继续优化：

- 卡片信息比较完整，但每张卡的层次还可以更利落
- “官方标题保留 / 中文说明 / 地点原文 / 来源可信度”已经具备，但可以更像成熟内容产品
- 列表节奏可以更有呼吸感，不要都像同一种卡片

### 动态列表页

- 现在逻辑清楚，但整体 still 比较“整齐卡片化”
- 可以加强“信息快读”的节奏感
- 原始标题和中文摘要之间的层级可以更明显
- 来源可信度可以更自然地融进去，而不是只靠 badge

### 详情页

活动详情页、动态详情页、明星页整体方向都对，但可以提升：

- 主标题区的气场
- 重点信息模块之间的节奏
- 关联内容模块的统一感
- 原文可搜索入口的可感知度

## 6. 我希望优化后的前台感觉像什么

不是普通 SaaS 后台感。  
也不是“AI 一键生成 landing page”感。  
更接近：

- 一个会做内容整理的中文编辑部入口
- 一个对新粉很友好的追星工具站
- 一个“既可信又不呆板”的服务型内容产品

关键词：

- 清楚
- 可信
- 有温度
- 不廉价
- 不花哨过头
- 不要模板味太重

## 7. 重点希望 Claude 优化的页面

优先级建议如下：

1. 首页 `/`
2. 活动日历页 `/calendar`
3. 动态列表页 `/news`
4. 活动详情页 `/events/[slug]`
5. 动态详情页 `/news/[slug]`
6. 明星页 `/stars/[slug]`

如果时间有限，请先把首页看透，再决定是否延伸到其他页面。

## 8. 我更希望看到什么类型的优化

### 希望看到

- 更强的首页主次关系
- 更清晰的内容分区
- 更成熟的卡片层级
- 更自然的“官方来源 / 已审核发布”可信度表达
- 更好的暗色界面呼吸感
- 更统一的标题、副标题、解释文本体系
- 更明确的行动入口

### 不希望看到

- 把所有卡片都改成同一种通用玻璃卡
- 盲目加渐变、加发光、加装饰
- 让文本更难读
- 用英文 UI 替代中文 UI
- 把“原文保留”弱化成不明显的附属信息
