# ThaiStar Bridge MVP Four-Week Plan

## Week 1
- Create the Next.js product shell and design system
- Define Supabase schema and seed data
- Ship homepage, calendar, star, event, news, services, and guide pages

## Week 2
- Connect Supabase reads and writes
- Implement the service request workflow
- Build the admin dashboard shell and moderation queues

## Week 3
- Add AI ingestion pipeline for scraping, translation, summary, and event extraction
- Add content review states and publishing workflow
- Improve search, filtering, and SEO metadata

## Week 4
- Final QA, mobile polish, analytics, and release checklist
- Backfill production content and admin playbooks
- Prepare Vercel deployment, env vars, and go-live runbook
