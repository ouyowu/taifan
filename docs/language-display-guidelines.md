# 泰娱桥语言展示规范

## 核心原则

- 界面、导航、流程说明：简体中文优先
- 专有名词、官方标题、地点名称：原文保留优先
- 展示方式：中文解释 + 原文保留，而不是把所有内容硬翻成中文

## 适合保留原文的内容

- 明星英文名
- 商场名
- 场馆名
- 活动官方标题
- 品牌名
- 主办方名
- 票务平台名

例如：

- `ICONSIAM`
- `Siam Paragon`
- `Paragon Hall`
- `Billkin`
- `PP Krit`
- `GMMTV`

## 适合中文优先的内容

- 网站导航
- 活动说明
- 买票步骤
- 预约活动说明
- 应援广告赞助说明
- 攻略与注意事项
- 摘要与内容整理

## 页面级规则

### 活动页

- 主标题：保留官方活动标题
- 副标题或摘要：用中文解释活动是什么
- 场馆与商场：保留原文写法，旁边提供中文理解信息

### 明星页

- 主标题：英文名或官方常用名
- 次标题：中文页面常用称呼
- 说明区：用中文解释艺人定位、适合关注什么内容

### 动态页

- 原始标题：保留官方标题
- 中文摘要：用中文快速解释发生了什么
- 正文整理：中文优先
- 原始来源：保留原链接与来源标签

## 一句话总结

界面中文化，专有名词原文化，操作说明中文强化。
